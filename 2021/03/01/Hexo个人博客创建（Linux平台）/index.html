搭建个人博客Hexo（CentOS 7）

什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

一、安装环境

  • Node.js
  • Git

二、Node.js安装

  1. 安装wget下载工具（如已安装，可跳过）

  2. 下载node.js最新的安装包，我个人存放在/usr/local/src路径下

  3. 解压安装包

  4. 查看版本，验证是否安装成功

  5. 重命名nodejs安装包所在路径

  6. 编辑/etc/profile文件，配置全局变量，方便全局使用

  7. 使环境变量生效，并验证

三、安装Git

  1. [root@lxfcentos7 /]# sudo yum install git-core

四、安装hexo

  1. [root@lxfcentos7 bin]# npm install -g hexo-cli

  2. 创建文件夹

  3. 初始化hexo

  4. 运行hexo（后台运行）

五、注意事项

  1. 运行hexo后，访问http://localhost:4000失败，需关闭linux服务器内的防火墙
